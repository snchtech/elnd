DECLARE
    l_selected_id NUMBER; -- Assuming the selected ID is stored in a variable
    l_title VARCHAR2(100); -- Assuming the title value will be stored in a variable

BEGIN
    -- Step 1: Retrieve the selected ID from the select list item
    l_selected_id := :P1_SELECT_LIST_ITEM;

    -- Step 2: Query the database to retrieve the title value
    SELECT title INTO l_title
    FROM your_table
    WHERE id = l_selected_id;

    -- Step 4: Assign the title value to another item
    :P1_ANOTHER_ITEM := l_title;
END;

1. Create a dynamic action for the select list item:
- Go to the page designer in Oracle Apex.
- Select the select list item.
- In the "Dynamic Actions" section, click on "Create" to create a new dynamic action.
- Choose the appropriate event for the dynamic action, such as "Change" or "Selection Change", depending on when you want the action to be triggered.
- Configure the necessary settings for the dynamic action, such as the selection type and affected elements.

2. Add a true action to the dynamic action:
- In the "True" branch of the dynamic action, click on "Add Action".
- Choose the action type "Execute PL/SQL Code".
- Paste the PL/SQL code mentioned earlier into the "PL/SQL Code" section.

3. Modify the PL/SQL code to update the actual title:
- In the PL/SQL code, replace `:P1_ANOTHER_ITEM` with the appropriate item name or ID where you want to display the actual title.
- You can use the `APEX_ITEM.SET_VALUE` function to set the value of the item dynamically. For example:
sql
     APEX_ITEM.SET_VALUE(p_idx => 'ITEM_NAME', p_value => l_title);
     

Replace `'ITEM_NAME'` with the actual name or ID of the item.

By setting up this dynamic action, the PL/SQL code will be executed every time the select list item's value changes. It will retrieve the corresponding title value and update the specified item with the actual title.
